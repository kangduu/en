## 学习英语的错误习惯

- 只背单词（要多练习）
- 一直不练习口语
- 没有快速学英语这一说
- 养成不好的英语习惯（中文发音记英语什么的）
- 忽略了那些在中文里没有的英文发音（th. V. dark L.）
- 学复杂的英语语法
  keep it simple！句子一定要有主语哦
- 不要担心犯错误
- 不给自己设目标
- 多学习英语（语言是技巧，需要多练习）
- 态度不行
